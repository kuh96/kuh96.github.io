JavaScript の正しい覚え方

万物は Map である

もちろん他に、数値、配列、いくつかの組み込みクラス・モジュール（Mathなど）、
いくつかの組み込みオブジェクト (Stringなど）があります。
関数オブジェクトについては後述。
